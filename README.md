# 🛒 CodeAlpha Django Ecommerce Store

A modern full-stack Ecommerce Store built using Django, HTML, CSS, and JavaScript. This project provides a complete online shopping experience with product browsing, cart management, wishlist functionality, customer reviews, and an admin dashboard.

---

## 🚀 Features

### 👤 User Features
- Browse products
- Product detail page
- Add products to cart
- Wishlist functionality
- Product reviews and ratings
- Dark/Light mode toggle
- Responsive UI

### 🛍️ Shopping Features
- Product catalog
- Product categories
- Product stock management
- Shopping cart
- Checkout page
- Order success page

### 📊 Admin Features
- Dashboard with statistics
- Add products
- Edit products
- Delete products
- Manage inventory

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Santhosh-nakka/CodeAlpha_Django_Ecommerce_Store.git
```

### Move into Project

```bash
cd CodeAlpha_Django_Ecommerce_Store
```

### Install Dependencies

```bash
pip install django
```

### Run Server

```bash
python manage.py runserver
```

### Open Browser

```text
http://127.0.0.1:8000/
```
